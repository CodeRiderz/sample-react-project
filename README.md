# SOCIO FrontEnd - ReactJs

## Installation
#### 1) Install node & npm
##### Debian/Ubuntu
```
sudo apt-get install -y nodejs
```
##### MacOS X
```
brew update && brew install node
```
#### 2) Install dependencies
```
npm install
```

## Run Application
#### 1) Run dummy service
```
npm run server
```
#### 2) Run WebpackDevServer (Dev)
```
npm start
```

## Build Application
```
npm run build


## Code Lint (Optional)
```
npm run lint
```
